Step-by-step instructions
1
Create a private GitHub repo
Go to github.com/new → name it (e.g. nbs-compliance-data) → set to Private → Create repository.
2
Create a Personal Access Token
Go to GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → Select scope: repo (full control) → Generate. Copy the token — it won't be shown again.
3
Enter details below
Paste your GitHub username, repo name, and token. Data will be saved as JSON files in the repo, one file per dealer per day.
